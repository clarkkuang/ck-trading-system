# Intel (INTC) 投资分析框架 — 第五监控实例

*设计日期 2026-07-23(Q2'26 财报当日盘后)。方法论:基率先行、[实据]/[推断]/[猜测] 标注、预承诺规则优先于相机决策。*

## 一、核心命题:一个二元期权,不是连续变量

Intel 与前四个实例根本不同。NVDA/NFLX 的核心是"增长降速多少"、AT&T 是"现金流转正兑现" —— 都是连续变量。**Intel 的核心是一个二元问题:Intel Foundry 能否成为可盈利的第三方先进代工业务** [推断]。

因此:
- **论点层**围绕代工里程碑(18A 盈利良率、14A 外部 firm 承诺、代工季亏收窄轨迹)+ 并行的产品端对 AMD 攻防,而非单一增长/估值指标。
- **估值层弃用 P/E**:GAAP 因 CHIPS 托管股按市值计价失真(Q2'26 净亏 -$11.0B / EPS -$2.16,几乎全来自 $12.5B 非现金损失 —— 股价越涨账面越亏);Non-GAAP 前瞻 P/E 在 ~$100 已达 ~60x。两种 P/E 都无法用 → 改用 **P/B + 分部估值(SOTP)** [实据]。
- **技术层弃用"回调买入"**:这是一只 $19→$142→~$100 的深度反转股,NVDA 式"上升趋势内回调买入"会被趋势门永久锁死 → 改用 **NFLX 式绝对价位阶梯 + 重上 200 日线=趋势修复** 的价值预承诺结构 [实据]。

## 二、决定性变量(单一)

> **Intel Foundry 在 18A-P/14A 先进节点上的外部已承诺晶圆量 / 外部代工营收。**

这一个变量决定代工是真实的第三方业务(牛,验证账面价值)还是补贴型内部成本中心(熊/尾,最终减值侵蚀支撑股价的账面)。数据可得性边界:Intel 不按季清晰单列外部代工营收(Q1'26 靠推算 ~$174M,Q2'26 未披露),故规则把可自动化触发点放在 `foundry_14a_firm_customers`(离散可观测)与 `foundry_op_loss`(季亏轨迹),外部营收本身列为人工 checklist + 支点叙事。

## 三、事实基础(Q2'26 财报,2026-07-23)[实据]

- 营收 $16.1B(+25% YoY,近 15 年最快);Non-GAAP EPS $0.42(市场预期 ~$0.21,超预期约翻倍);Non-GAAP 毛利率 41.8%
- **GAAP 净亏 -$11.0B / EPS -$2.16 —— 几乎全来自 $12.5B CHIPS 托管股按市值计价的非现金损失**(政府 ~10% 持股,股价涨→托管负债估值升→账面亏),属会计噪声非经营恶化;Non-GAAP 净利 $2.2B
- 分部:DCAI $6.26B(+59%,创纪录,主引擎);CCG $8.88B(+13%);Intel 产品合计 $15.14B(+28%,经营利润 $4.82B)
- **Intel Foundry:营收 $5.77B(+31%,绝大部分内部),经营亏损 -$2.09B —— 连续收窄(Q4'25 -2.5 → Q1'26 -2.4 → Q2'26 -2.09),朝 2027 盈亏平衡在轨**;外部营收仍极小(Q1'26 ~$174M)
- 资本开支上调至 2026 全年 $20B+(原"与去年持平"),Q2 调整后 FCF **-$8.4B**;股息仍暂停
- 现金及短投 $29.7B,总债务 $50.5B;盘后涨超 13%,YTD +170% 但 7 月回落 ~28%

## 四、估值:P/B + SOTP [实据+推断]

- **P/B**:现 ~4.6x,BVPS ~$22.18(总权益 ~$125B / 5.03B 股)—— **Intel 历史最高**,远超 2020 垄断期 ~2.8x、2025 困境期 ~0.8x
- **粗 SOTP**(5.03B 股):产品业务(FY25 经营利润 ~$12.7B)按 12-18x ≈ $150-240B;Mobileye ~$7B;Altera 49% ~$4.3B;减净债 ~$12B;代工为摆动项(熊 ~$0-30B / 基准 ~$40-80B / 牛 ~$150-300B)→ **熊 ~$33 / 基准 ~$50 / 牛 ~$92 每股**
- **关键结论**:~$100 现价已在牛市 SOTP(~$92)之上,期权溢价已全额支付;硬资产地板落在 **$20-25**(政府 $20.47 成本 + Nvidia/SoftBank ~$23 + 账面 $22),对 $100 的持仓几乎无近端硬价格保护 —— 下行保护主要是"国家冠军隐性兜底"(政府 ~$44B 敞口),属政治/需求性软底

## 五、情景带(12-18 个月)

| 情景 | 概率 | 隐含价 | 假设 |
|---|---|---|---|
| 牛/尾部上行 | 15% | $170 | 18A-P/14A 外部承诺转量产、AI 周期延续,P/B 重估至 5-6x → "美国版 TSMC" [猜测] |
| 基准 | 35% | $95 | AI 顺风延续但代工仍未证明,贴近街共识 ~$108 有部分回吐 [推断] |
| 熊 | 35% | $57 | AI capex 消化 + P/B 压缩至 2-3x,代工亏损磨底、14A 承诺落空 [推断] |
| 尾部下行 | 15% | $30 | 代工减值 + AI 转向 + 稀释,回到 1-1.5x 账面;政府/战略软底托住难破 $20 [推断] |

**概率加权公允 ≈ $83** vs 现价 ~$100 → **现价较框架公允价高约 20%**,且高于街共识乐观、已站上牛市 SOTP 与创纪录 4.6x P/B。

## 六、框架初始判定:卖出观察区 / 择机减仓

**HOLD-核心 + TRIM-into-strength,不在现价加仓** [推断]。当年"便宜期权 + SOTP 地板"的入场逻辑已基本兑现,非对称性反转为"定价充分的动量股 + 价值陷阱尾部"。基率诚实提示:约 80% 半导体项目错过初始进度;大型"国家冠军"反转完全重估约 **1/3-1/4**(Nokia/Nortel 为失败模板;AMD 反转靠**走 fabless** —— 与 Intel 重资本代工路线相反)。**代工成功非基准情形**,下行至 base/bear($45-90)空间大于上行至 bull($150-190)。

新增买入应门控于"代工外部营收/14A 承诺"支点变量与 **$72/$55/$40** 价位阶梯,而非现价追高。

## 七、三层信号体系

**技术层(NFLX 式阶梯 + 趋势修复)** —— 给分批价位
- 买(仅论点未破):T1 ≤$72(价值回归带上沿)/ T2 ≤$55(逼近 base-SOTP)/ T3 ≤$40(逼近软底)。有意无视趋势门。
- 趋势修复:重上并连续 2 周站稳 200 日线 → dip 动量系统恢复可用(为"真跌破后再夺回"预置)
- 卖(advisory):连续 4 周收于 200 日线下方 = 动量破位降杠杆;论点/估值卖点优先

**估值层(P/B)** —— 给安全边际
- 买:P/B ≤ 2.0(回到历史便宜期权区)—— 现 ~4.6x 远高于此
- 卖:P/B ≥ 5.0(创纪录、高于牛市 SOTP)= optionality 全额计价,趋势性减仓

**论点层(代工二元 + 产品侵蚀)** —— 给方向
- 买:14A 外部 firm 大客户 ≥1(最硬向上触发,现为 0)/ 18A 盈利良率 ≥70% / 代工季亏持续收窄
- 卖:代工季亏 QoQ 重新走阔 / DCAI 增速连续 2 季减速(AI 周期见顶)/ AMD 服务器份额 ≥50% / FCF 连续 2 季为负 / 18A 年底仍 <70% 良率(CFO 红线)

## 八、已知边界

- **200 日线诚实边界**:因准抛物线上涨,现 200 日线远低于现价 —— 此刻 ~33% 回撤发生在 200 日线**上方**,故趋势修复短期不触发,趋势破位需先真跌破。价位阶梯是近端可用机制。
- **GAAP 全面弃用**:CHIPS 托管按市值计价噪声会反复出现,所有盈利/估值以 Non-GAAP + 分部经营利润 + P/B/SOTP 为轴。
- **18A 良率数据边界**:官方从未披露具体良率数字(press 估计 ~55-65%)。故 `yield_18a_profitable_pct` 字段**不预填估计值**(否则会误触发 sell)—— 靠 checklist 人工跟踪 CFO "2026 年底" 红线,待官方 Q4 数字再录入。
- **拆分不是估值地板**:$20 认股权证(挂钩代工持股 51% 门槛)是毒丸,封堵"卖代工"的 SOTP 变现路径。
- 与页 07 dip 系统的关系:阶梯有意无视趋势门(价值预承诺 vs 动量),遵循 [docs/signal_semantics.md](signal_semantics.md) 的分层优先级。
